# iRAWShare Public Tracker

This is a public repository for a part of iRAWShare.

While iRAWShare itself is currently closed-source, this repository contains the `Localizable.xcstrings` file used for localization.

Translation contributions, bug reports, feature requests, compatibility reports, and general feedback are welcome.

## Limitations
The source code of iRAWShare is not included in this repository.

This repository is intended for:

- Localization and translation contributions
- Bug reports
- Feature requests
- Compatibility reports
- General feedback

Thank you for helping improve iRAWShare.
